Instalación y configuración

  1. Descargamos la versión NOOBS de
    http://www.raspberrypi.org/downloads

  2. Descargamos el programa de
    https://www.sdcard.org/downloads/formatter_4/eula_windows/

  3. Instalamos el programa SDFORMATTERv4 y lo ejecutamos

  4. Seleccionamos "FORMAT SIZE ADJUSTMENT" en OFF en el apartado Option

  5. Le damos a Format, y en Aceptar

  6. Esperamos a que se termine el proceso. Importante, no desconectar la unidad mientras se formatea

  7. Copiamos los archivos dentro de la sd

  8. Insertamos la sd, y realizamos todas las conexiones(HDMI, y corriente).
    Luego se nos inicia, y elejimos Raspbian, y le damos instlar

  9. Si nos sale una advertencia le decimos que sí

  10. Dejamos que se instale

  11. Una vez instalado, si no ah habido problemas, nos saldrá como que todo está instalado correctamente

  12. Se nos inicia el siguiente menú. Elejimos el numero 3

  13. Elejimos iniciar seción grafica con el usuario pi y reiniciamos

  14. Abrimos consola como root, y añadimos el grupo profesor

  15. Abrimos consola como root, y añadimos el usuario profesor

  16. Abrimos consola como root, y añadimos el usuario profesor. Le ponemos contraseña al profesor

  17. le ponemos contraseña al root

  18. Le ponemos contraseña a Pi

  19. Mirarmos la MAC
